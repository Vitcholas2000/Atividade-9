O que é um Form?

Um Form é um container usado para agrupar controles responsáveis pela entrada de dados, ou simplificando, input do usuário. Pode ser caracterizado como uma views com scroll, que possui controles estáticos como uma tela de configurações com botões toggle e áreas de texto.

Container Form

"Container Form", é um conjunto de Forms que são agrupadas a partir de um container, ganhando a tag "Container Form".

    • O que é "Post"?

    "Post", é um metódo de envio de dados de um Forms para o servidor WEB, usando o metódo Post para a requisição HTTP, significa que são enviados de forma anônima pro servidor, são normalmente usados para logins mais particulares, informações de senhas, cartãos(formas de pagamentos), etc...
